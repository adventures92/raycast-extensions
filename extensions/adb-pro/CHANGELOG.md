# ADB Pro Changelog

## [1.0.0] - 2025-12-22

### Added
- **Device Management**: Comprehensive list and control for physical and virtual devices.
- **App Management**: Fast actions for Force Stop, Clear Data, Uninstall, and App Settings.
- **Wireless Connectivity**: Streamlined interface for connecting to devices over Wi-Fi.
- **Real-time Logcat**: Built-in log viewer with filtering and search capabilities.
- **System Settings Toggles**: Quick access to Wi-Fi, Mobile Data, Dark Mode, Airplane Mode, and Layout Bounds.
- **Developer Utilities**: Permissions manager, Network Proxy settings, Resolution/DPI changer, and Battery/Key simulation.
- **Advanced Tools**: Mirror screen capability, Clipboard sync, and Benchmark utilities.
